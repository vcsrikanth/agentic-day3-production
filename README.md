# Agentic Day 3 Production

Support agent application with LangGraph and OpenAI.

**Important:** The `.env` file is not in version control — do not commit it (it contains secrets).

## Setup

1. Create a `.env` file with `OPENAI_API_KEY=your-key`
2. Install dependencies: `pip install -r requirements.txt`
3. Run: `python app.py`
